static array parseParameters(string $query, string|callable|null $replaceParams = null, string $regex = '/(:[a-z]+)|\\?|\\$\\d+/i')

Parses a query containing parameters into an array, and can replace them with a predefined replacement (can be a callable).

The callable is used to return numbered parameters (such as used in PostgreSQL), or any other kind of parameters supported by the DBMS.

Parameters

string $query
string|callable|null $replaceParams If null is passed, it will not replace the parameters.
string $regex

Return Value

array [ 'query' => string, 'parameters' => array ] The parameters array is an numeric array (= position, starting at 1), which map to the original parameter.
