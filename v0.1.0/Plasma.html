Interfaces

ClientInterface
The client interface for plasma clients, responsible for creating drivers and pooling.
ColumnDefinitionInterface
This interface defines a common column definition abstraction.
CommandInterface
The basic interface for commands. Some drivers may extend this interface to provide additional functionalities.
DriverFactoryInterface
A driver factory is used to create new driver instances. The factory is responsible to create the drivers with the necessary arguments.
DriverInterface
The minimum public API a driver has to maintain. The driver MUST emit a close event when it gets disconnected from the server.
QueryResultInterface
This is just a basic interface. There is an additional interface which defines that the query result is stream-based.
QueryableInterface
Any queryable class (can execute queries) implements this indirectly or directly.
StatementInterface
Represents any prepared statement.
StreamQueryResultInterface
This is the more advanced query result interface, which is a readable stream.
TransactionInterface
Transactions turn off auto-commit mode and let you rollback any changes you have done during it.
