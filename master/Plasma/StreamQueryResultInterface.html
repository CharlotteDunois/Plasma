StreamQueryResultInterface

interface StreamQueryResultInterface implements QueryResultInterface, ReadableStreamInterface (View source)

This is the more advanced query result interface, which is a readable stream.

That means, for SELECT statements a data event will be emitted for each row. At the end of a query, a end event will be emitted to notify of the completion.
